목표 시스템 기능 구성도
사용자 대면 주요 기능

손글씨 문제 풀이

Canvas 기반 손글씨 입력 및 이미지 업로드, Mathpix API 연동 고정밀 수식 인식(선형대수 특화), 실시간 편집/정답판정, YOLO 기반 수식 영역 탐지, AI 사고력 유도형 피드백 제공.

파이썬 실습 환경

선형대수 문제 연동 코드 자동 생성(NumPy/SciPy), 웹 기반 코드 작성/실행(React/Docker), 정적/동적 코드 분석 및 LLM 기반 오류 수정 피드백.

챗봇 기반 학습 지원

OpenAI GPT & LangChain 기반 대화형 지원(선형대수/파이썬), 소크라테스식 단계적 힌트, CoT/Few-Shot Learning, RAG Agent(수업자료 기반 답변).

유저 가이드 (RAG 챗봇)

시스템 기능 관련 자연어 질문 응답, 사용자 매뉴얼/학습문서 벡터 검색 기반 답변 생성, 운영 유연성 확보.

학습자 지원 및 관리

대시보드 학습 통계

개인 맞춤형 학습 현황 시각화(풀이 수, 정답률), 게이미피케이션 요소(랭킹, 경험치), Chart.js 기반 분석으로 자기주도 학습 지원.

교과과정 지원 LMS

SmartLead LMS 연동, 주차별 학습 내용/목표 제공, 단원별 연습문제(상세정보 포함), 학사 시스템 연동(수강정보, 일정).

사용자 ID 관리

한림대 통합정보시스템 SSO 연동, 로그인/아웃 로그 관리, 사용자 정보 관리(열람/수정/삭제), OAuth 2.0 기반 보안 인증.

사용자 피드백 수집

웹/앱 내 '피드백 보내기' 기능, 입력 내용 관리자 이메일 자동 전송, 서비스 품질 향상 및 사용자 만족도 개선에 활용.

데이터 관리 및 활용

사용자 데이터셋 구축

Caliper Analytics 표준 기반 데이터셋, 주요 상호작용(질문, 프롬프트, LLM응답) 단계별 저장, 메타데이터 포함(JSON-LD).

DB 설계 요구사항

통합 기반 논리적 구조(중복 최소화, 일관성), 유연성/확장성 고려, 데이터 정합성/무결성 확보, 성능 최적화 설계.

DB 모니터링 및 안정화

이관 데이터 검증/품질관리, 실시간 모니터링 및 자동 통보, 오류 대응(보정/재이관), 정기 백업 및 접근 제어.

데이터 품질 및 표준 관리

품질관리 체계 수립(목표/정책), 데이터 표준 정의(명명규칙/형식), 구축/운영 단계별 품질 확보, 구조/값 검증.

시스템 기반 및 보안

가드레일 기능

LLM 오용 방지(오류응답, 편향/유해 콘텐츠 차단), 입력/후처리 가드레일, 외부 가드레일 연동(Meta Llama Guard 등).

시스템 아키텍처 및 개발

공개SW 우선 활용/관리, 다국어 지원(i18n), 클라우드 환경 구성, 컴포넌트 기반 설계, 표준 개발(React/NestJS/Docker).

소프트웨어 개발 보안

시큐어 코딩(XSS/SQLi 방지), 인증/권한 제어(OAuth2/SSO/RBAC), 정적분석/보안코딩 규칙, 민감정보보호/로그관리.

시스템 확장성 및 UI

수평확장(Scale-out) 아키텍처, Auto Scaling, 무중단 배포, 응답속도 2초 이내 목표, 동시접속 1천명+ 수용, 웹 기반 반응형 UI.
